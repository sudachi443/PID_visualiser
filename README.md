# PID_visualiser
ブラウザ上で動作するPID制御の直感的なシミュレーターです。
物理法則に基づく物体の挙動を通じて、P（比例）、I（積分）、D（微分）の各制御成分がどのようにシステムに影響を与えるかを視覚的に学習できます。  
## Credits　
GitHub Copilot および Gemini モデルを使用しています。
## 特徴  
- 対話的なシミュレーション: リアルタイムでPIDパラメータを調整し、挙動の変化を確認可能。
- 制御モードの比較: ON/OFF制御、P制御、PD制御、PID制御の切り替えが可能。
- 外乱のシミュレーション: 風（定常的な外乱）や突風（瞬間的な外乱）に対するシステムの耐性を検証可能。
- グラフ描画: 物体の位置と目標値の推移を軌跡として描画。
## 使い方
1. ブラウザで直接開くか、ローカルサーバーで実行してください。  
2. 左側のコントロールパネルから「制御モード」を選択します。  
3. 必要に応じて「PID パラメータ」を調整し、物体の挙動を観察してください。  
4. 赤い線（目標値）はマウスドラッグで自由に移動可能です。
